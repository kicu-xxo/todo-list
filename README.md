## 📣기능📣

1.  input에 할 일을 적고 엔터키를 누르면 리스트에 추가됩니다.
2.  할 일이 10개 이상 생기면 스크롤 바가 생성됩니다.
3.  하트❤ 아이콘

    -누르면 중요 표시

4.  오른쪽 체크✔ 아이콘

    -누르면 완료 표시

5.  오른쪽❌ 아이콘

    -누르면 해당 할 일 삭제

6.  input 아래 All Clear🗑 아이콘

    -누르면 모든 할 일 삭제
